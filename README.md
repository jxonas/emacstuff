
Emacstuff
=========

Emacs customizations.

## Instalation

1. Install Emacs
2. In your terminal/console:

  ```bash
  cd ~
  git clone https://github.com/jxonas/emacstuff.git
  mv emacstuff .emacs.d
  cd .emacs.d
  git submodule init
  git submodule update
  cd org-mode
  make autoloads
  ```
3. Open Emacs
4. Wait packages instalation
5. Close Emacs
6. Open Emacs, that's it!
